Bracketed terminal command — `[ LABEL ]`, fills with inverse video on hover. Use for every action.

```jsx
<Button variant="primary" icon="▸" onClick={go}>Accéder</Button>
<Button variant="secondary">Retour</Button>
<Button variant="danger" block>Déconnexion</Button>
```

Variants: `primary` (peak) · `secondary` (dim) · `danger` (blinks). Props: `block`, `icon`, `disabled`. Monochrome — no color, ever.
