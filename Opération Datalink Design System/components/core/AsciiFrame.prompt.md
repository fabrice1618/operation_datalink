The signature terminal group-box — a square frame with the title cut into the top border as `┤ TITLE ├`. The primary container of the whole system.

```jsx
<AsciiFrame title="Dossier de mission" right="03:42">
  <p>…</p>
</AsciiFrame>
```

`title` and `right` both interrupt the top border. `bodyStyle` overrides the inner padding.
