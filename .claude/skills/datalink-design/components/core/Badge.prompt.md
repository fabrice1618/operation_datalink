Terminal status tag — monochrome state. Pass short text (`OK`, `··`, `!!`, a count).

```jsx
<Badge variant="success">OK</Badge>
<Badge variant="pending">··</Badge>
<Badge variant="danger">ERR</Badge>
```

Variants: `success` (inverse block) · `info` (bright) · `warn`/`danger` (blink) · `pending` (faint).
